# Statische Offline-PWA ohne zentralen Server

Die Anwendung wird als statisch auslieferbare PWA gebaut und speichert den Datenbestand lokal auf einem einzelnen Tablet, weil der Schuetzenstand im Wald liegt und Internetzugang dort nicht verlaesslich ist. Wir verzichten bewusst auf zentralen Server, Benutzerverwaltung und Mehrgeraete-Synchronisation; Export und Backup/Restore decken die notwendige Datenuebergabe ab.
